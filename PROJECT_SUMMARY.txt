PROJECT OVERVIEW
================================================================================
Gemator is a proof-of-concept web application for translating Korean manhwa 
speech bubbles using Google's Gemini AI. Features an interactive HTML5 Canvas 
interface for manual bubble selection with real-time translation.

READY TO RUN INSTRUCTIONS
================================================================================

1. Prerequisites:
   - Python 3.8+
   - Gemini API key (from https://makersuite.google.com/app/apikey)

2. Setup (First Time):
   cd "/Users/leonardcl/Documents/00 Personal/Workstation/coding/gemator"
   source venv/bin/activate
   # Already done: python3 -m venv venv
   # Already done: pip install -r requirements.txt
   # Already done: cp .env.example .env
   
   # REQUIRED: Add your API key to .env file:
   # Edit .env and replace: GEMINI_API_KEY=your_gemini_api_key_here

3. Run Application:
   source venv/bin/activate
   python app.py
   
4. Open Browser:
   http://localhost:5000

5. Usage:
   - Upload a Korean manhwa image (PNG/JPG, <10MB)
   - Click and drag rectangles around speech bubbles
   - Click "Translate" button
   - View English translations alongside original

================================================================================

- Manual bubble selection required

- Basic text fitting (no advanced typography)
- Temporary file storage (uploads deleted after translation)
- No automatic bubble detection (manual selection only)
